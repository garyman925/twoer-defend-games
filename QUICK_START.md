# 🚀 快速啟動指南

## 問題解決

如果你遇到了 `Module not found: Error: Can't resolve './src'` 錯誤，這是因為Webpack配置的問題。我已經為你修復了這個問題。

## 🔧 修復內容

1. **修復了Webpack配置** - 現在正確指向 `./src/core/Game.js` 作為入口點
2. **創建了HTML模板** - 移到根目錄，包含遊戲容器
3. **添加了佔位場景** - 所有場景都已創建，避免導入錯誤
4. **修復了配置常數** - GameConfig中的缺失常數已補充
5. **創建佔位資源** - 避免資源載入錯誤

## 🎮 現在可以啟動遊戲了！

```bash
# 確保在項目根目錄
cd C:\xampp\htdocs\games\tower-defend

# 啟動開發服務器
npm run dev
```

## 🌐 訪問地址

遊戲將在以下地址自動打開：
- **http://localhost:3000**
